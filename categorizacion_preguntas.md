- Pregunta 3: Seguridad del código en IA generativa

- Pregunta 7: Políticas a nivel de organización
- Pregunta 8: API REST de Suscripciones Business
- Pregunta 9: Feedback en GitHub Mobile
- Pregunta 10: Comando /fix
- Pregunta 11: Exclusión de datos del entrenamiento

- Pregunta 14: Configuración de detección de duplicados
- Pregunta 15: API de métricas de uso
- Pregunta 16: GitHub Copilot en CLI

## Dominio 3: Cómo funciona GitHub Copilot y controla los datos (15% - ~6 preguntas)
- Pregunta 17: Limitaciones de sugerencias de código
